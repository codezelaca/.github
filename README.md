# Codezela Career Accelerator organisation profile

This repository powers the public GitHub organisation profile for Codezela Career Accelerator.

GitHub treats a public organisation repository named .github as special. The README inside the profile folder becomes the organisation profile page that everyone sees.

Location of the public profile README  
profile/README.md

## What this repository is for

1. Store and maintain the public organisation profile content
2. Keep shared organisation level community files in one place
3. Provide consistent contribution rules across CCA repositories
4. Make it easy for maintainers to update the profile without touching product or curriculum repos

## Repository layout

1. profile/README.md  
   The organisation profile content shown on the GitHub organisation page

2. CONTRIBUTING.md  
   Contribution rules used across repositories

3. CODE_OF_CONDUCT.md  
   Behaviour standards for the community

4. SECURITY.md  
   How to report security issues safely

5. ISSUE_TEMPLATE/ and PULL_REQUEST_TEMPLATE.md  
   Templates to keep contributions clean and easy to review

If some of these files are not present yet, they can be added later as the organisation grows.

## How to update the organisation profile

1. Open profile/README.md
2. Make your changes with clear wording and accurate information
3. Keep formatting clean and easy to scan
4. Open a pull request with a short summary of what changed and why
5. Request review from a maintainer

## Content standards for the profile README

1. Keep it career focused and outcomes driven
2. Prefer clarity over buzzwords
3. Mention real tracks, real standards, real tools
4. Keep links current and working
5. Use UK English spelling and tone
